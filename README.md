# Study Box (学習の箱) 📦

「動けばいい」コードから「メンテナンスしやすい綺麗なコード」への進化を目指しています。

FastAPI + WebSocket をベースにした、攻防一体型の学習用チャットアプリ。
- **Status**: Phase 2 (DB連携・セキュア認証) 完了

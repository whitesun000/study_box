# Study Box (学習の箱) 📦

Pythonを中心とした、日々の学習コードや思いついたアイデアをストックしていくリポジトリです。

## 🎯 目的
- プログラミング（主にPython）の基礎スキルの向上。
- 実用的な自作ツール（ディレクトリ構成取得ツール、パスワード管理ツール等）の開発。
- 綺麗なコード（モジュール化、カプセル化）を意識したリファクタリングの練習。
- 過去の挫折（Webアプリ、ゲーム解析）を糧にした再挑戦の記録。

## 🛠 現在の開発・学習テーマ

- [ ] **GitHub連携**: 学習成果の定期的なプッシュと公開。

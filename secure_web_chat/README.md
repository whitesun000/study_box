# Secure Web Chat Ecosystem

学習用の「攻防一体型」チャットアプリケーションです。
単なるチャット機能だけでなく、セキュリティの脆弱性と、それに対する堅牢な実装を体系的に学習することを目的としています。

## 🛠 現在の進捗
- [x] **フェーズ1**: FastAPI + WebSocket によるリアルタイム通信の基盤実装
- [x] **フェーズ2**: SQLite + プリペアドステートメントによる認証の基礎
- [x] **フェーズ3**: パスワードのハッシュ化（bcryptによる保存の安全性向上）
- [x] **フェーズ4**: セッション管理（Cookieによるログイン状態の維持とログアウト）
- [ ] **フェーズ5**: セキュリティ・ラボ（SQLインジェクションの攻防体験モード）
- [ ] **フェーズ6**: AI監視エンジンによるリアルタイム攻撃検知
- [ ] **フェーズ7**: ネットワークトラフィックの可視化（Scapy等の活用）

## 📂 ディレクトリ構成
クリーンコードとモジュール化を意識し、役割ごとにフォルダを分離しています。
- `app/core/`: 
    - `auth.py`: ログイン・ログアウト等の認証ロジック
    - `security.py`: パスワードハッシュ化等の安全管理
    - `database.py`: DB接続管理
    - `websocket_handler.py`: チャット通信制御
- `app/templates/`: UI（HTML）
- `static/`: フロントエンド資産
    - `css/`: デザイン（base, nav, login, styleに細分化）
    - `js/`: フロントエンド制御（WebSocket通信等）
- `data/`: データベースファイル（`.gitignore` により除外）

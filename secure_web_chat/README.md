# Secure Web Chat Ecosystem

学習用の「攻防一体型」チャットアプリケーションです。
単なるチャット機能だけでなく、セキュリティの脆弱性と、それに対する堅牢な実装を体系的に学習することを目的としています。

## 🛠 現在の進捗
- [x] **フェーズ1**: FastAPI + WebSocket によるリアルタイム通信の基盤実装
- [x] **フェーズ2**: SQLite + プリペアドステートメントによる認証の基礎
- [x] **フェーズ3**: パスワードのハッシュ化（bcryptによる保存の安全性向上）
- [x] **フェーズ4**: セッション管理（Cookieによるログイン状態の維持とログアウト）
- [x] **フェーズ5**: セキュリティ・ラボ（SQLインジェクションの攻防体験モード） ✅ DONE
- [ ] **フェーズ6**: AI監視エンジンによるリアルタイム攻撃検知（← 次はここ！）
- [ ] **フェーズ7**: ネットワークトラフィックの可視化

## 🚀 アップデート内容 (ver 0.5)
- **セキュリティ・ラボ**: 
    - ボタン一つで「脆弱モード（SQLIが通る）」と「セキュアモード」を切り替え可能。
    - ログイン成功/失敗や、攻撃の検知をリアルタイムに表示する「疑似ログウィンドウ」を実装。
- **認証バイパスのシミュレーション**: 
    - `' OR '1'='1` 等のクエリを用いて、パスワードなしでログインする挙動を再現。
    - 不正なログインを検知すると、ユーザー名を自動的に `Hacker` に書き換える演出を追加。

## 📂 ディレクトリ構成
- `app/core/`: 
    - `auth.py`: 認証ロジック（脆弱モードの切り替え・SQLI判定を含む）
    - `security.py`: パスワードハッシュ化
    - `database.py`: DB接続管理
    - `chat.py`: WebSocket通信制御
- `app/templates/`: UI（HTML）
- `static/`: フロントエンド資産
    - `css/`: デザイン（モジュール化済み）
    - `js/`: 
        - `main.js`: 基本的な通信制御
        - `lab.js`: セキュリティ・ラボのログ・演出制御（New!）
- `data/`: データベースファイル

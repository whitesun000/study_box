# Secure Web Chat Ecosystem

学習用の「攻防一体型」チャットアプリケーションです。
単なるチャット機能だけでなく、セキュリティの脆弱性と、それに対する堅牢な実装を体系的に学習することを目的としています。

## 現在の進捗
- [x] **フェーズ1**: FastAPI + WebSocket によるリアルタイム通信の基盤実装
- [x] **フェーズ2**: SQLite + プリペアドステートメントによるセキュアな認証の実装
- [ ] **フェーズ3**: パスワードのハッシュ化（保存の安全性向上）
- [ ] **フェーズ4**: セッション管理（Cookie/JWT）の実装
- [ ] **フェーズ5**: AI監視エンジンによるリアルタイム攻撃検知
- [ ] **フェーズ6**: Scapyを活かしたネットワークトラフィックの可視化

## ディレクトリ構成
クリーンコードとモジュール化を意識した構成を採用しています。
- `app/core/`: 認証、DB接続、WebSocket処理などの基幹ロジック
- `app/templates/`: UI（HTML）
- `app/static/`: デザイン・フロントエンド制御（CSS/JS）
- `data/`: データベースファイル（GitHub除外対象）

# direct_info (Project Content Extractor)

プロジェクトのディレクトリ構造とファイルの中身を抽出し、一つのログファイルにまとめるGUIツールです。

## 概要
指定したフォルダ内をスキャンし、フォルダ構成のツリーとソースコードの内容をテキストとして抽出します。
AI（LLM）にプロジェクトの全体像を読み込ませたい時や、リファクタリング前のコード整理、プロジェクト構造のドキュメント化に最適です。

## 特徴
- **直感的なGUI操作**: フォルダ選択や保存先の指定をマウス操作で行えます。
- **階層構造の可視化**: フォルダ構成をインデント付きのツリー形式で出力します。
- **柔軟な抽出オプション**: ファイルの中身を書き出すかどうかをチェックボックスで切り替え可能です。
- **スマートな除外設定**: `.git`, `__pycache__`, `.venv` などの不要なディレクトリは自動的にスキップします。
- **幅広い対応形式**: `.py`, `.txt`, `.html`, `.css`, `.js`, `.php`, `.md`, `.json` などのテキストファイルを自動抽出します。

## 使い方
1. `main.py` を実行してGUIを起動します。
2. 「スキャン対象フォルダ」を選択します。
3. 「ログ保存先」を確認（または変更）します。
4. ファイルの中身を含める場合はチェックを入れ、「スキャン開始」をクリックします。

## ディレクトリ構成
```text
direct_info/
├── main.py              # エントリーポイント
├── gui/
│   └── app.py           # GUI画面構成・イベント制御
└── utils/
    └── file_processor.py # ファイル走査・書き出しロジック
```

- Tkinter(標準ライブラリ)

# dhirect_info

プロジェクト内のディレクトリ構造とファイルの中身を抽出し、一つのログファイルにまとめるツールです。

## 概要
このツールを実行すると、指定したフォルダ内のファイルをスキャンし、ソースコードをテキストとして抽出します。AIにコードの全体像を把握させたい時や、プロジェクトの構造を確認したい時に便利です。

## 特徴
- **階層構造の可視化**: フォルダ構成をツリー形式で出力します。
- **コード抽出**: `.py`, `.txt`, `.html`, `.css`, `.php` などのテキストファイルを自動で読み込みます。
- **除外設定**: `.git` や `venv` などの不要なフォルダは自動的にスキップします。
- **ユーザー指定**: 実行時にスキャンしたいフォルダを選択可能です。

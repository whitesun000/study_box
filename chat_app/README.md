# Python 暗号化チャットツール

このプロジェクトは、Pythonのソケット通信、暗号化ライブラリ（Cryptography）、およびTkinterを使用した、学習用のセキュアなチャットアプリケーションです。

## 特徴
- **エンドツーエンド暗号化**: `Fernet`（AES）を利用し、通信経路上のメッセージを保護します。
- **セキュリティモニタリング**: 短時間の連打（DoS対策）や不審な記号（SQL注入試行の検知）をスコア化し、自動ブロックします。
- **モジュール化設計**: 
  - `core/`: 通信、暗号化の基幹ロジック。
  - `gui/`: 見た目（Layout）と制御（Controller）を分離。
  - `security/`: 不正アクセス検知。

## フォルダ構成
```text
chat_app/
├── core/
│   ├── encryption.py     # 暗号化・復号クラス
│   ├── server_logic.py   # サーバー側通信ロジック
│   └── client_logic.py   # クライアント側通信ロジック
├── gui/
│   ├── layout.py         # GUIの配置・見た目
│   └── controller.py     # GUIとロジックの橋渡し
├── security/
│   └── monitor.py        # 不正検知・ブラックリスト管理
├── main.py               # CUI版（テスト用エントリポイント）
└── gui_main.py           # GUI版（統合コンソール）
```

# セットアップ
## 必要なライブラリ
仮想環境を有効にした状態で、暗号化ライブラリをインストールしてください。
```
pip install cryptography
```

# 使い方
## 1. サーバーの起動
gui_main.py を実行し、**「サーバー起動」**ボタンをクリックします。
```
python chat_app/gui_main.py
```
※ サーバーが起動するとターミナルに [*] サーバー起動: 9999 と表示されます。

## 2. クライアントの参加
同じウィンドウ（または新しく立ち上げた gui_main.py）で、**「クライアントとして参加」**ボタンをクリックし、名前を入力します。

## 3. メッセージの送受信
下部の入力欄にメッセージを入力し、Enterキーまたは「送信」ボタンで会話が始まります。

## 4. 使い方（GUI版）
    1. 起動: ターミナルで python chat_app/gui_main.py を実行。

    2. サーバー開始: ウィンドウの「サーバー起動」ボタンを押します。

    3. 参加: 「クライアントとして参加」を押し、名前を入力します。

    4. チャット: 下部の入力欄からメッセージを送信します。
       ※ 他のPCや別のターミナルから参加する場合は、サーバーのIPアドレスに合わせて controller.py を調整してください。
